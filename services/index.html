Services

Trois exemples de blocs animés différemment : fade-up, slide-right et text-fade-mask, combinés sur une seule page pour tester les interactions entre animations.

Direction artistique & motion design

Conception de systèmes d'animation cohérents, pensés pour le scroll autant que pour la navigation entre pages.

Chaque interaction est calibrée pour rester perceptible sans ralentir la lecture du contenu.

  • Motion design
  • Scroll UX

Développement Astro & GSAP

Intégration de View Transitions natives, scripts modulaires, et respect strict du cycle de vie du DOM pour éviter toute fuite mémoire en production.

Accessibilité du mouvement

Toutes les animations sont déclenchées par des transformations CSS performantes (opacity, transform), facilement désactivables via prefers-reduced-motion dans une itération ultérieure.

Fin de page

Remontez en haut de la page pour observer les animations de sortie (reverse) se déclencher dans l'ordre inverse.
